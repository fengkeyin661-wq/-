# 每周验收与运营看板指标

## 周验收清单（建议每周固定一次）

### A. 功能验收
- [ ] 医生端提交随访后，用户端“基础身体指标”在实时或10秒内更新
- [ ] 用户端“我的随访记录”展示最新执行单（含下次随访时间）
- [ ] 用户端提醒策略（D-7/D-3/D0）生效且不重复提醒
- [ ] 用户端自主更新后，医生端可读到同一字段值

### B. 数据一致性验收
- [ ] 抽样 30 个用户，核心字段一致率 >= 99%
- [ ] 空值/异常单位比率 <= 1%

### C. 质量验收
- [ ] `npx tsc --noEmit` 通过
- [ ] 核心文件 lint 无新增错误
- [ ] 无高优先级回归缺陷

## 看板指标（按周）

### 过程指标
- 随访到达率 = 完成随访人数 / 计划随访人数
- 执行单完成率 = 完成任务人数 / 下发执行单人数
- 提醒触达率 = 触发提醒用户 / 应提醒用户

### 结果指标
- 风险降级率（RED->YELLOW/GREEN, YELLOW->GREEN）
- 关键指标改善率（SBP/DBP/GLU/Weight/TC/TG/LDL/HDL）

### 质量指标
- 同步成功率
- 同步延迟 P95
- AI 回退率（非模型成功生成）

## 建议周报模板
1. 本周上线项
2. 指标变化（过程/结果/质量）
3. 异常与根因
4. 下周优化项
